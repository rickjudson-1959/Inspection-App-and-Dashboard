## PART 8: TROUBLESHOOTING & SUPPORT

### Chapter 23: Common Issues & Solutions

23.1 User Issues
| Issue | Cause | Solution |
|-------|-------|----------|
| Can't log in | Password expired | Reset via admin |
| Missing tickets | Wrong date range | Adjust filter |
| App not syncing | Connectivity | Check internet, force sync |
| Photos not uploading | Storage full | Clear device storage |

23.2 Data Issues
| Issue | Cause | Solution |
|-------|-------|----------|
| $0 calculated costs | Missing rate | Add to rate table |
| Variance shows incorrectly | Name mismatch | Standardize naming |
| Missing inspector match | Date/foreman mismatch | Manual investigation |

23.3 Process Issues
| Issue | Cause | Solution |
|-------|-------|----------|
| Invoice total doesn't match | Missing/extra tickets | Reconcile before finalizing |
| Can't find archived ticket | Wrong search | Use invoice number search |
| Dispute not resolving | Missing info | Request specific documentation |
