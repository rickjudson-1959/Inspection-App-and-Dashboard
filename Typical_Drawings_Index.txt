# Typical Drawings Index
# Project: Pipeline Construction
# Purpose: This index allows AI search to find specific drawings by title, number, or description

## How to Use This Index
When an inspector asks about a specific drawing, crossing type, or construction detail,
search this index to find the relevant drawing number and page. The inspector can then
access the drawing in the Typical Drawings section of the Project Document Vault.

## Usage Instructions for AI Agent

When an inspector asks about a drawing:
1. Search this index for matching keywords or drawing numbers
2. Return the drawing number, title, and page number
3. Direct the inspector to access the Typical Drawings in the Project Document Vault
4. The page number corresponds to the page in the Typical Drawings PDF

Example queries this index helps answer:
- "What's the drawing for a road bore crossing?" → TYP-1002, Page 4
- "Show me erosion control details" → TYP-1029 through TYP-1033
- "What's the typical for a watercourse crossing?" → TYP-1000 (Bore) or TYP-1001 (Open Cut)
- "Where can I find ditch details for farmland?" → TYP-1014, Page 16
- "What's the drawing number for cathodic protection test stations?" → TYP-1022, Page 24
